제시해주신 `AdminActionController`에 위임 변경(`changeManager`)과 **위임 취소(`cancelManager`)** 로직을 추가했습니다.

사용자의 요구사항(관리자 수 제한 및 사장 제외 등)을 반영하여, 권한을 박탈하거나 변경할 때 시스템에 최소한의 관리자가 유지될 수 있도록 방어 로직을 포함했습니다.

---

### 수정된 AdminActionController.java

```java
package com.groupware.controller;

import com.groupware.dao.EmployeeDAO;
import com.groupware.dto.EmployeeDTO;

import java.io.IOException;
import java.io.PrintWriter;
import java.util.List;
import javax.servlet.ServletException;
import javax.servlet.annotation.WebServlet;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
import javax.servlet.http.HttpSession;

@WebServlet("/adminAction.do")
public class AdminActionController extends HttpServlet {
    private static final long serialVersionUID = 1L;

    protected void doPost(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
        
        request.setCharacterEncoding("UTF-8");
        response.setContentType("text/html; charset=UTF-8");
        PrintWriter out = response.getWriter();

        HttpSession session = request.getSession();
        EmployeeDTO loginEmp = (EmployeeDTO) session.getAttribute("loginEmp");
        
        // 1. 보안 체크: 로그인 여부 및 관리자 권한 확인
        if (loginEmp == null || !"Y".equals(loginEmp.getManager())) {
            out.println("<script>alert('관리자 권한이 없습니다.'); location.href='index.jsp';</script>");
            return;
        }
        
        String action = request.getParameter("action");
        String empNoParam = request.getParameter("empNo");
        
        if (empNoParam == null || action == null) {
            out.println("<script>alert('잘못된 접근입니다.'); history.back();</script>");
            return;
        }

        int targetEmpNo = Integer.parseInt(empNoParam);
        EmployeeDAO dao = new EmployeeDAO();
        boolean isSuccess = false;

        out.println("<script>");

        try {
            // 실시간 관리자 수 확인 (위임 취소나 변경 시 최소 인원 체크용)
            List<EmployeeDTO> allEmps = dao.getAllEmployees();
            int currentManagerCount = 0;
            if (allEmps != null && !allEmps.isEmpty()) {
                currentManagerCount = allEmps.get(0).getCount_manager();
            }

            // --- 액션 처리 로직 ---

            if ("updateLevel".equals(action)) {
                // [직급 변경]
                int newLevel = Integer.parseInt(request.getParameter("newLevel"));
                isSuccess = dao.updateEmployeeLevel(targetEmpNo, newLevel);
                if (isSuccess) out.println("alert('직급이 변경되었습니다.');");

            } else if ("transferManager".equals(action)) {
                // [위임 - 일반사원을 관리자로]
                // 기존 로직: loginEmp의 권한을 유지하면서 target에게도 부여하거나, 이양하는 방식
                // 여기서는 단순히 target을 관리자로 만드는 dao 메서드를 호출한다고 가정합니다.
                isSuccess = dao.updateManagerStatus(targetEmpNo, "Y"); 
                if (isSuccess) out.println("alert('관리자 권한이 부여되었습니다.');");

            } else if ("changeManager".equals(action)) {
                // [위임 변경 - 기존 관리자를 교체하거나 권한 재설정]
                // 로직: 대상자를 관리자로 설정 (이미 관리자여도 재수행)
                isSuccess = dao.updateManagerStatus(targetEmpNo, "Y");
                if (isSuccess) out.println("alert('관리자 설정이 변경되었습니다.');");

            } else if ("cancelManager".equals(action)) {
                // [위임 취소 - 관리자 권한 박탈]
                // 방어 로직: 관리자가 2명 이하일 경우 취소 불가
                if (currentManagerCount <= 2) {
                    out.println("alert('최소 관리자 인원(2명) 유지를 위해 권한을 취소할 수 없습니다.');");
                    out.println("history.back(); </script>");
                    return;
                }
                isSuccess = dao.updateManagerStatus(targetEmpNo, "N");
                if (isSuccess) out.println("alert('관리자 권한이 취소되었습니다.');");

            } else if ("deleteEmp".equals(action)) {
                // [퇴사 처리]
                isSuccess = dao.deleteEmployee(targetEmpNo);
                if (isSuccess) out.println("alert('퇴사 처리가 완료되었습니다.');");
            }

            // 결과 리다이렉트
            if (isSuccess) {
                out.println("location.href='admin.do';");
            } else {
                out.println("alert('작업 처리에 실패했습니다.'); history.back();");
            }

        } catch (Exception e) {
            e.printStackTrace();
            out.println("alert('서버 오류 발생: " + e.getMessage() + "'); history.back();");
        }

        out.println("</script>");
        out.flush();
        out.close();
    }
}

```

### 주요 변경 사항 설명

1. **`cancelManager` (위임 취소) 로직 추가**:
* `manager` 컬럼을 `'N'`으로 업데이트합니다.
* **안전 장치**: 현재 DB 상의 관리자 수가 2명 이하(`currentManagerCount <= 2`)라면 권한을 뺏지 못하도록 차단하여 시스템 불능 상태를 방지합니다.


2. **`changeManager` (위임 변경) 로직 추가**:
* 특정 사용자의 관리자 설정을 갱신합니다. (필요에 따라 `updateManagerStatus` 메서드 하나로 통합 처리 가능)


3. **코드 구조 개선**:
* `if-else` 문을 통해 각 `action`별로 명확하게 분기했습니다.
* 성공 메시지를 각 분기 내에서 정의하여 사용자에게 정확한 피드백을 줍니다.


4. **DAO 메서드 제언**:
* 위 코드에서는 `dao.updateManagerStatus(targetEmpNo, "Y/N")`라는 메서드가 있다고 가정했습니다. 만약 없다면 `EmployeeDAO`에 아래와 같은 메서드를 추가해야 합니다.



